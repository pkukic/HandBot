# HandBot
For automating jobs which can be done with robot hands - MATLAB-based robot control with kinematics and computer vision

## Requirements
- MATLAB with the following toolboxes:
  - Robotics System Toolbox
  - Computer Vision Toolbox
  - Simulink
  - Dynamixel SDK (for hardware control)

## Usage
1. Open MATLAB
2. Navigate to this directory
3. Run `robot_control_app.mlapp` for the GUI interface
4. Or open the `.slx` files in Simulink for simulation
